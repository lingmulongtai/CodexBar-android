# CodexBar for Android v0.3.6

This stable release makes Codex account linking easier to complete and adds quota-pace coaching to the dashboard.

## Codex account connection

- Keeps the device code visible in CodexBar instead of opening the browser immediately.
- Adds a copy-first flow and removes separators from the clipboard value so the full code can be pasted into split code fields.
- Opens the OpenAI sign-in page only after the user explicitly taps the browser button.
- Pins the device authorization endpoints to `https://auth.openai.com` with regression coverage.
- Replaces raw hostname failures with localized guidance for Private DNS, VPN, ad-blocker, and network issues.

## Usage pace coaching

- Calculates average quota use per hour from the provider's full reset cycle when the cycle duration is available.
- Shows used quota versus elapsed cycle time, target-pace multiplier, forecast usage at reset, and projected reserve.
- Adds plain-language guidance such as room to use more, good pace, slightly fast, and overusing.
- Shows a positive message when a successful Codex response includes a long-term limit but no short-term five-hour limit.
- Falls back safely when a provider omits or returns an invalid window duration.

## Verification

- Passed 131 unit tests with no failures.
- Passed Android Lint and the debug APK build with strict dependency verification.
- Confirmed the copy-first Codex flow on a separate Android 16 / API 36 emulator without completing a real account login.
- Completed a focused read-only review of the authentication and quota changes and fixed the reported duration-overflow edge case.

## Install or update

1. Download `app-release.apk` from the assets below.
2. Open it on the Android device and allow installation from the browser or file manager if Android asks.
3. Install it over the current version. Do not uninstall first if you want to keep linked accounts and settings.

Release assets are signed with the existing production certificate and include `SHA256SUMS`, a CycloneDX SBOM, build provenance, and GitHub artifact attestations.
