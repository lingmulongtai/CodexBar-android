# CodexBar for Android v0.3.7

This stable release fixes the Codex and GitHub Copilot device-code connection failures reported after v0.3.6 and adds clearer, safer setup instructions.

## Codex account connection

- Retries transient DNS resolution failures during both device-code polling and the final OAuth token exchange.
- Keeps retrying at the provider interval until the originally issued sign-in code expires instead of abandoning a completed browser login after one temporary resolver failure.
- Preserves immediate failure handling for HTTP authentication errors and non-DNS network failures.
- Adds regression coverage for DNS recovery after browser authorization and during token exchange.

## GitHub Copilot account connection

- Requests JSON explicitly from both GitHub Device Flow endpoints.
- Fixes the `Unexpected JSON token` failure caused when GitHub returned URL-encoded `device_code=...` data to the JSON-only Retrofit client.
- Restores the one-time code card so users can copy the code before explicitly opening GitHub's sign-in page.
- Adds request-contract tests for the JSON header, endpoint paths, form fields, and decoded responses.

## Guided setup and installation

- Adds a numbered four-step account-link guide directly to the Settings screen in English and Japanese.
- Shows the official sign-in host before browser navigation and explains when to return to the app for on-device validation.
- Explains paste-ready codes, expiry recovery, and safe handling of short-lived sign-in codes and provider tokens.
- Expands the README with release APK installation, SHA-256 verification, temporary installer permission cleanup, provider-specific sign-in hosts, and troubleshooting.

## Verification

- Passed 135 unit tests across 51 suites with no failures.
- Passed Android Lint and the debug APK build with strict dependency verification.
- Verified the GitHub Device Flow request and JSON response contract with MockWebServer.
- Verified matching English and Japanese resources and the complete in-app instruction order.

## Install or update

1. Download `app-release.apk` from the assets below.
2. Optionally compare its SHA-256 digest with the included `SHA256SUMS` file.
3. Open the APK on the Android device and allow installation from the current browser or file manager if Android asks.
4. Install it over v0.3.6. Do not uninstall first if you want to keep linked accounts and settings.
5. Turn off the temporary **Install unknown apps** permission after installation.

Release assets are signed with the existing production certificate and include `SHA256SUMS`, a CycloneDX SBOM, build provenance, and GitHub artifact attestations.
